# Challenge 07 (Hacktiv8 FGA)

## Deskripsi Project
Project ini berfokus dalam pembuatan REST API untuk penyimpanan data-data buku menggunakan framework *Gin* dan *PostgreSQL* sebagai database. Dalam API ini memiliki 5 method yaitu:
- Get All Books
- Get Book by ID
- Create Book
- Update Book
- Delete Book
